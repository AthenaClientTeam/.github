 - **High-Performance Utility & QoL Client for Minecraft Fabric 1.21.11**

*   **Advanced Performance:** Custom FPS throttling (Unfocused FPS Cap) to save system resources while tabbed out.

*   **Smart QoL:** Built-in utilities designed for seamless 1.21.11 gameplay.
*   **Lightweight Core:** Minimal overhead, ensuring your hardware gives the best performance.
